# Prerequisites
1. zsh
2. stow

# Steps
1. clone the repo `git clone https://github.com/chuakid/dotfiles.git --recurse-submodules --depth 1`
2. `./install.sh`
3. make a .gitconfig.local file with the [user] directive with name and email


# Additional
1. stow mpv if needed
